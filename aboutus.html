About us
 

The main objective of this project is to shop the groceries online, so that making the user to shop the house hold items online without any effort of manually going to the outlet. This application is very useful because it provides the automation of shopping addition to that delivery of the items to the house is done which makes the user friendly. Application gives the wide variety of house hold items so that user can choose the items and cart it and order it online, this application also gives the online payment facility and manual payment as the goods or items delivered to the house. This system is capable of maintaining details of various customer, vendors, products and storing all the day to day transactions such as generation of shipment address bills, handling customer and product receipts, updating of store the central concept of the application is to allow the customer to shop virtually using the internet and allow customers to buy the items and articles of their desire from the store.


The end user of this product is a departmental store where the application which hosted on the web and the administrator maintains the database. The application which is deployed at the customer database, the details of the items are brought forward from the database for the customer view based on the selection through the menu and the database of all the products are updated at the end of each transaction.
